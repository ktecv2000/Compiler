##File Hierarchy
###ProblemName/
>例:problemA/ 存放problemA的題目。

###ProblemName/Ans/
>存放已通過此題DOMjudge測試的CODE，檔名記得加上自己的名子。

###ProblemName/TestData/
>存放此題在DOMjudjge上的公開測資。

###Temp/
>個人CODE暫存。

###Share/
>其他分享資料\工具。
